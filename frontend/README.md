# LearnWithAI Frontend

Vue 3 + TypeScript + Vite + Tailwind CSS

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Vue 3 Composition API
- TypeScript
- Vue Router
- Pinia State Management
- Tailwind CSS with Digital Garden theme
- WebSocket for AI chat
- Responsive design
